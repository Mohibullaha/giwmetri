What is event bubble?

Bubbles in the event interface indicate read-only properties. It checks whether the event has bubbled through the DOM tree or not.

What is callback function?

A callback function is one that is passed to another function as an argument and then called inside the outer function to complete some routine or action. And this is the call back function.
